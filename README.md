# Rural Service Finder

## 📌 Project Description
This project helps rural users find essential services like doctors, electricians, plumbers, etc.

## 🎯 Objective
To provide a centralized platform for accessing local services easily.

## 🛠 Technologies Used
- HTML, CSS
- Java (JSP & Servlet)
- MySQL
- Apache Tomcat

## ⚙️ How It Works
- User selects a service
- System fetches data from database
- Displays available providers

## 🔮 Future Scope
- Mobile app integration
- GPS location tracking
- Online booking system
- AI-based service recommendation
